# [olca-core](https://github.com/GreenDelta/olca-modules/tree/master/olca-core) 
This module, developed by the openLCA team, contains all the core tools and functionalities (duh!) required to carry out an LCA analysis for any given product/process. It is made up of 6 parts:

## 1. Core Java Objects and Methods: [`org.openlca.core`](https://github.com/GreenDelta/olca-modules/tree/master/olca-core/src/main/java/org/openlca/core)

## 2. Geo ? : [`org.openlca.geo`](https://github.com/GreenDelta/olca-modules/tree/master/olca-core/src/main/java/org/openlca/geo)

## 3. JSONLD ? : [`org.openlca.jsonld`](https://github.com/GreenDelta/olca-modules/tree/master/olca-core/src/main/java/org/openlca/jsonld)

## 4. Text ? : [`org.openlca.text`](https://github.com/GreenDelta/olca-modules/tree/master/olca-core/src/main/java/org/openlca/text)

## 5. Utilities ? : [`org.openlca.util`](https://github.com/GreenDelta/olca-modules/tree/master/olca-core/src/main/java/org/openlca/util)

## 6. Validation ? : [`org.openlca.validation`](https://github.com/GreenDelta/olca-modules/tree/master/olca-core/src/main/java/org/openlca/validation)
